Detailed Description

An observer to be passed to an se_solver via its solve method. Its purpose is to store the sequence of guesses for the values of the unknown quantities, along with additional information, e.g., whether each new guess was made using the get_next_guess (adjustment_made == false) or adjust_bad_guess (adjustment_made == true) methods.
