Detailed Description

This module determines tissue mass derivatives based on the output of either 'partitioning_growth_calculator' or 'no_leaf_resp_partitioning_growth_calculator'. While this module will function using an adaptive step size integrator, some aspects of it may not work as intended (e.g. the check for negative rhizome mass).
