Calculates the change in value of the unknown quantities given a vector of initial values.

To solve the set of equations, we seek a set of initial values that produces a vector of zeroes when used as an input to this function.

Parameters
[in]unknown_quantity_vectora vector of initial values for the unknown quantities
[out]difference_vectora vector of (final - initial) values for the unknown quantities
