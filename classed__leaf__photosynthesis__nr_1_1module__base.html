Detailed Description

A general module representing photosynthesis calculations where the simultaneous equations are solved using the Newton-Raphson method. Note that this module cannot be created by the module_wrapper_factory since its constructor has a different signature than a typical module. Instead, it must have concrete derived classes. This class has been made abstract to indicate this. See the base class for more information. Note also that this class does have a do_operation method defined, so base classes (mostly) just need to specify which modules to use.
