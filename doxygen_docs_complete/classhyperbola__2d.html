Detailed Description

Represents two 2D hyperbolas defined by f(x,y) = 1 + 1 / (x + y) and g(x,y) = 1 + 1 / (2x - y). Intended to be used as a simple test case for simultaneous equation solvers. One solution to f(x,y) = x, g(x,y) = y exists at x = -1.126529, y = 0.656279 (determined numerically).
