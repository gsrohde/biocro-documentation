Detailed Description

Module describing photosynthesis at the leaf level using the default sub-modules defined by ed_leaf_photosynthesis::reference_module_names. The Newton-Raphson method is used to find self-consistent values for stomatal conductance, boundary layer conductance, net assimilation, and leaf temperature.
