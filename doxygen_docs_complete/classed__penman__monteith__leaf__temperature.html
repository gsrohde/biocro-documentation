Detailed Description

Uses the Penman-Monteith equation to determine leaf temperature (see eq. 14.11e from Thornley (1990), p. 418). Currently only intended for use by Ed.

See the "ed_nikolov_conductance_forced" module for a discussion about converting molar conductances (with units of mol / m^2 / s) to energy conductances (with units of m / s). In short, we replace all conductances g in the Thornley formula with g * volume_per_mol_of_an_ideal_gas.
